* Qdrant Example App

An example application show-casing [[https://qdrant.tech/qdrant-vector-database/][Qdrant's vector database]].

** Requirements
- [[https://docs.docker.com/compose/][Docker Compose]]
- [[https://github.com/casey/just][just]]

** Bring the server up 
#+begin_src sh
just up
#+end_src

** Run the example 
#+begin_src sh
just example 
#+end_src
